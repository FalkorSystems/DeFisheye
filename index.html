DeFisheye

This tool remove distortion from videos shot with a fisheye lens. Great for processing GoPro video footage.

Usage

DeFisheye is quite simple to use. Operation is the same on both Mac and Linux. There are two ways to operate DeFisheye.

Drag and Drop

  • Drag your MP4 file onto the DeFisheye icon.
  • A dialog will pop up with a dropdown box allowing you to select the camera, resolution, and FOV that you used for your video.
  • Select the correct camera, resolution, and field of view.
  • If you'd like, you can choose a different place to save the output file. We only support .ogg on Linux and .avi on Mac
  • Click 'OK'

Manual

  • Launch DeFisheye
  • Use the file selector to choose the MP4 file you want to DeFisheye
  • A dialog will pop up with a dropdown box allowing you to select the camera, resolution, and FOV that you used for your video.
  • Select the correct camera, resolution, and field of view.
  • If you'd like, you can choose a different place to save the output file. We only support .ogg on Linux and .avi on Mac
  • Click 'OK'

FAQ

What if I am using a different resolution or field of view?

Just ask us and we'll perform a calibration for the settings you'd like to use.

What if I am using a different camera?

You can use the gopro_calibration tool that we have created to help you calibrate your own camera. Or you can ask that we perform the calibration for you. If we have that camera/lens combination, we'll be happy to do it, but if not you will have to lend us yours.
